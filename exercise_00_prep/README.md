# Exercise 0 — Preparation

**Theme: Cost of Producing Intelligence**

This app demonstrates how AI can reduce the cost of knowledge delivery by providing conversational answers tailored to a specific audience.

## Purpose

Use this app to understand the baseline concept of AI-powered chat interfaces before exploring more specialized applications in the following exercises.

## Features

- **Chat Interface**: Enter any question or topic to discuss
- **Audience Customization**: Adjust the response style for different audiences (e.g., students, experts, executives)
- **Plain Language Output**: Responses include concise answers with brief explanations

## Try It

1. Ask about a concept you're learning
2. Request an explanation of a technical term
3. Explore how changing the audience affects the response

## Run

```bash
streamlit run exercise_00_prep/app.py
```
